Runs on WOW64 on 64 bit machines because Mapwindow 4.8 requires a seperate build for its 64 bit version.
This means you need to install 32 bit MapWindow on any 64 bit machine to run this plugin.

You will want to use Inno Setup to modify the installation file. (*.iss)

The Code Name, which follows Microsoft Naming conventions is: OpenNspect
The Public Name, used in all error messages and dialogs should be OpenNSPECT
