Nonpoint-Source Pollution and Erosion Comparison Tool (N-SPECT) version 1.5.1

Required Software:
1. ArcGIS 9.3
2. Spatial Analyst extension

Installation Instructions:
1. Run NSPECT_v.1.0.1_install.exe.  Follow the instructions to complete the setup and an "NSPECT" directory will be installed on your C:\ drive.
2. Run NSPECTInstall.v151.ArcGIS 93.msi.  This file installs a patch on top of version 1.0.1.  Follow the instructions to complete the installation and N-SPECT will be updated for ArcGIS 9.3.
